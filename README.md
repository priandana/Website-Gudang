# Warehouse Spreadsheet Hub

Landing page untuk menampilkan link Google Spreadsheet, dibagi 3 kategori:
- Umum
- Finishgood
- Material

## Jalankan Lokal
```bash
npm install
npm run dev
```

## Edit Link Spreadsheet
Ubah data di `lib/sheets.ts`.

## Deploy ke Vercel
Push ke GitHub lalu import project di Vercel (framework: Next.js).
